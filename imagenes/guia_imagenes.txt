INSTRUCCIONES GENERALES
────────────────────────────────────────────────────────────────────────────────────

  · Todas las imágenes deben colocarse en esta carpeta: /app2/imagenes/
  · Formato recomendado: WebP (el sistema también acepta JPG, PNG, JPEG)
  · Si entregas JPG o PNG, el sistema las convertirá automáticamente a WebP
    al cargarlas desde el módulo de Productos (botón "Subir imagen")
  · Resolución entrada recomendada: 400×400px a 800×800px
  · El sistema recorta automáticamente a 180×180px al guardar
  · Fondo recomendado: blanco o transparente
  · Imágenes limpias, sin marcas de agua

  NOMENCLATURA DE ARCHIVOS:
  · Usa el mismo código del producto: CAB001.webp, AUD001.webp, etc.
  · Mayúsculas o minúsculas, el sistema no distingue
  · Sin espacios ni caracteres especiales

DÓNDE CONSEGUIR LAS IMÁGENES
────────────────────────────────────────────────────────────────────────────────────

  1. Sitios oficiales de las marcas (DJI, Logitech, Sony, JBL, TP-Link)
     → imágenes de prensa con fondo blanco de alta calidad

  2. Amazon.com → buscar el modelo exacto, las imágenes del seller oficial
     suelen estar en fondo blanco → clic derecho → Guardar imagen

  3. Google Shopping → buscar "producto fondo blanco" → Imágenes
     → filtrar por licencia si es para uso comercial

  4. Unsplash.com / Pexels.com → para fotos generales de tecnología

  5. Freepik.com → vectores e imágenes de productos tech (licencia gratuita
     con atribución)

  NOTA: Para uso demo/interno cualquier imagen basta.
  Para uso comercial/público, asegurarse de usar imágenes con licencia libre.
